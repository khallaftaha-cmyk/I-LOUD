About us:

I-LOUD est bien plus qu'un club ; c'est un incubateur de talents spécialisé dans l'architecture Cloud et la culture DevOps. Nous comblons le fossé entre la formation d'ingénieur à l'INPT et les exigences de l'industrie (AWS, Azure, GCP). De la virtualisation à l'orchestration de conteneurs, nous propulsons nos membres vers des certifications de haut niveau et des projets complexes, car chez I-LOUD, nous croyons que l'ingénierie n'a de limite que le ciel.

Cloud Notes API

A fully cloud-native RESTful API for creating, organizing and syncing notes across devices. Built with a microservices architecture and deployed on AWS with auto-scaling and high availability in mind.

JWT authentication with refresh token rotation and role-based access control Real-time sync via WebSockets — changes reflect instantly across all connected clients Full-text search powered by Elasticsearch with tag and category filtering Deployed on AWS ECS with RDS PostgreSQL and S3 for attachment storage

Post Management API

A scalable content management API designed for multi-platform publishing. Handles post lifecycle — drafting, scheduling, publishing and archiving — with a powerful webhook system for third-party integrations.

Multi-tenant architecture supporting isolated workspaces per organization Scheduled publishing with a cron-based job queue using Bull and Redis Webhook engine that notifies external services on every post state change CI/CD pipeline with GitHub Actions deploying to Azure Container Apps

Cloud-IoT Monitor & Digital Twin API

An advanced IoT monitoring platform that creates live digital twins of physical devices on the cloud. Sensor telemetry streams in real-time, enabling predictive analytics, anomaly detection and remote control of connected hardware.

Digital twin engine mirrors physical device state with sub-second latency via MQTT Time-series telemetry stored in InfluxDB with Grafana dashboards for visualization ML-based anomaly detection alerts operators before hardware failures occur Deployed on GCP with Pub/Sub for event-driven device communication at scale
